### Install Requirements:

-  npm run install

### Run:

#### Gerar Tabuleiro

-  node index.js --generate

#### Imprimir o tabuleiro gerado

-  node index.js --print

#### Executar o método

-  node index.js --run

#### Executar o método com um tabuleiro expecifico

-  node index.js --run --board=filename.json
